4. Inverse Design — Multi-Output Weighting

flowchart TB subgraph Weighting["Per-Target Weight Calculation"] IV["Inverse Variance: w_var = 1 / var(target)"] CP["Constraint Precision: w_prec = data_range / constraint_span"] Combine["w = w_var x w_prec"] Norm["Mean-normalise so avg weight = 1.0"] IV --> Combine CP --> Combine Combine --> Norm end subgraph Penalty["Weighted Penalty Function"] Violation["violation = predicted - bound"] Scale["Normalise by data range"] Weight["Multiply by target weight"] Sum["total = Sum of weight x (violation / range)^2"] Violation --> Scale --> Weight --> Sum end Norm --> Weight
Weight examples:
• Tight constraint hardness 49–51 (span=2) over data range 500 → high w_prec
• Low-variance target → high w_var
• One-sided constraint (only min or max) → neutral w_prec = 1.0
